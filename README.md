# KonradGroupTest
Assessment for Konrad Group  
Built using the MLB Api:  
- Notifications.json
- Master_Scoreboard.json
- Scoreboard.json

#Running The Web app
Git clone it  
npm install the required packages  
npm start  
"Favourite team" defaults to Blue jays!
